# cdut-wqhelper

成都理工大学问渠学堂浏览器扩展（Chrome/Edge Manifest V3）。

**功能**：课件 PPT 批量导出、课堂笔记提取（AI 摘要 / 语音识别 / 热词云）、课程列表快速入口。

## 安装

1. 下载源码或 Release 包解压
2. 浏览器访问 `chrome://extensions/`，开启「开发者模式」
3. 点击「加载已解压的扩展程序」，选择项目目录
4. 打开 [问渠学堂](https://wqxt.cdut.edu.cn/) 即可使用

## 使用

| 页面 | 点击扩展图标 | 功能 |
|------|------------|------|
| 课程目录页 | 📦 批量下载 | 勾选课节 → 📄 导出PPT / 📝 导出笔记 |
| 课节回放页 | 📄 单节下载 | 📄 导出PPT / 📝 导出笔记 |
| 首页/我的学习 | 📋 我的课程 | 课程列表 → 进入课程目录 |

**设置**（⚙ 齿轮按钮）：
- 课件下载方式：自动/翻页
- 笔记导出：是否包含语音识别

## 安全

- 本扩展仅在你已登录的问渠学堂页面内运行
- 不收集、不传输任何个人数据
- API 调用依赖你的浏览器登录态（cookie），无硬编码凭据
- 全部代码在本地浏览器中执行，无远程服务器
