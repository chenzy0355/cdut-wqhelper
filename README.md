# 问渠学堂课件导出插件 (WQT PPT Downloader)

本项目是一个用于问渠学堂（Web 端）的浏览器扩展程序。通过接管网页前端的翻页逻辑与资源监听，将视频课程中的幻灯片图像缓存并重新排版，最终输出为无边距的 PDF 格式文件，便于离线查阅与批注。

## 功能特性

* **状态锁定制**：引入双击确认机制，避免误触导致脚本在非目标元素上执行。
* **异步静默缓存**：模拟 DOM 点击事件自动遍历幻灯片，并通过 Performance API 实时监听资源加载状态。
* **自定义视图排版**：重写 `@page` 媒体查询规则，强制 16:9 物理页面比例（256mm × 144mm），消除常规 A4 打印产生的边缘留白。
* **非侵入式预览**：数据提取完成后，自动构建独立的 HTML 视图在新标签页进行渲染，不破坏原视频播放页面的 DOM 结构。

## 安装说明

本项目目前仅提供源码包安装，适用于基于 Chromium 内核的浏览器（如 Google Chrome, Microsoft Edge）。

1. 下载本项目的 `.zip` 压缩包并解压到一个固定目录。
2. 在浏览器地址栏输入并访问扩展管理页：`chrome://extensions/` （Edge 浏览器为 `edge://extensions/`）。
3. 在页面右上角（或左下角）开启 **开发者模式**。
4. 点击 **加载已解压的扩展程序**，选择刚才解压的文件夹。

## 使用指南

1. 进入问渠学堂视频播放页面，等待页面基础资源加载完成。
2. **点击回放页面右侧的 PPT 放大按钮，切入幻灯片主导视图。**
3. 点击浏览器扩展栏中的本插件图标，页面顶部将出现运行状态指示栏。
4. 根据指示栏提示，在网页下方找到并 **连续点击两次** `下一页 (>)` 按钮，以指定遍历目标。
5. 脚本将自动接管翻页动作。请等待页面右上角的进度提示，直至缓存完成。
6. 缓存结束后，点击指示栏生成的 **在新标签页预览并导出** 按钮。
7. 在新标签页中点击打印，目标打印机选择 **另存为 PDF**，并务必在更多设置中将“边距”调整为 **无**。
